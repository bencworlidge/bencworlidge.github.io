Spotify Track Analysis

A React App which utilises the Spotify API to find and return any song's Tempo, Key, Energy, Danceability and Acousticness.

Spotify Track Analysisclose

As a musician, I'm always needing to know the key and BPM of a track quickly and easily. Utilising OAuth and Node.js, this app connects to the Spotify API and allows you to search for any song within it's database. Once the song has been selected, the relevant information is returned.
